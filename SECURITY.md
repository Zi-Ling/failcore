# Security Policy

FailCore is an execution-time safety runtime for AI agents.
Security issues are taken seriously, especially those involving
sandbox escapes, policy bypasses, or unintended side effects.

---

## Supported Versions

Only the **latest released version** of FailCore is supported with security fixes.

### What to include

Please provide as much of the following as possible:

- A clear description of the vulnerability
- Steps to reproduce (or a minimal PoC)
- The affected version(s)
- Potential impact (e.g. sandbox escape, policy bypass, filesystem/network access)

### What to expect

- You will receive an initial response **within 72 hours**
- If the report is accepted, we will work on a fix and coordinate disclosure
- If the report is declined, we will explain why

We appreciate responsible disclosure and security research.
Thank you for helping make FailCore safer.
